Part 1 Documentation

The image above shows a an example of what happens when you connect to this app. All the code and files used for this app are contained inside the checkVisit folder. To connect to this app, either press the link in the sidebar, or you can directly connect to it by entering the URL http://localhost:8000/part1/

Upon connecting to this app, the main1.js script is run (located in the static folder). In this script, we will create a cookie on the domain called counter. Everytime the user connects to the app, this counter is incremented. That is how we are able to keep track of the number of times clients have visited the webpage. There are also methods in this file to find the ipadress and timezone of the client.
